# Lab 11

## Kubectl Secrets

### Create

```shell
> kubectl create secret generic lab11-secret --from-literal=API_KEY=hello
secret/lab11-secret created
```

### Verify

```shell
> kubectl get secrets
NAME           TYPE     DATA   AGE
lab11-secret   Opaque   1      24s
```

### Decode

```shell
> kubectl get secret lab11-secret -o jsonpath='{.data}'
{"API_KEY":"aGVsbG8="}
```

```shell
$echo "aGVsbG8=" | base64 -d
hello
```
